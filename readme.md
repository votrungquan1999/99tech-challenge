# 99Tech Challenge

## Problem 1: Sum to N

Three implementations of computing sum of first n natural numbers.

```bash
cd problem1
bun install
bun run index.ts
```

## Problem 2: Token Swap Interface

Next.js application with token swap functionality.

```bash
cd problem2
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in browser.

## Problem 3: Component Refactoring

React component code review identifying computational inefficiencies and anti-patterns.

**Files:**
- `someComponent.tsx` - Original component
- `fixedComponent.tsx` - Refactored component
- `analysis.md` - Issues identified and fixes applied

## Requirements

- Bun runtime
